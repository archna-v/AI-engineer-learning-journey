Day 1: AI & Machine Learning Fundamentals

1. Data Collection

Data can be collected using: • Web Scraping
• APIs
• Existing Databases

Key Principle

Garbage In → Garbage Out

The quality of data directly affects model performance.

2. Types of Data

Labeled Data

✓ Contains predefined labels or annotations

Example:

Dog → "Dog"

Cat → "Cat"


Advantages:

High accuracy

Suitable for supervised learning

Better evaluation


Unlabeled Data

✓ No predefined labels

Advantages:

Easy to collect

Lower cost

Useful for pattern discovery


3.Metadata

Metadata is information that describes other data.

Examples:

Author

Date Created

File Size

Location Information


4. Machine Learning

Machine Learning enables systems to learn patterns from data and make predictions.

Types of Machine Learning

Supervised Learning

Uses labeled data

Learns from input-output pairs


Unsupervised Learning

Uses unlabeled data

Finds hidden patterns


Reinforcement Learning

Learns through rewards and penalties

Commonly used in robotics and gaming

Day 3: NLP, LLMs & Transformers

1. Natural Language Processing (NLP)

NLP enables computers to understand and generate human language.

Applications

Translation

Summarization

Question Answering

Chatbots


2. Language Models

Masked Language Models

Predict missing words using surrounding context.

Autoregressive Models

Predict the next word using previous words.

3. Large Language Models (LLMs)

Capabilities:

Text Generation

Coding Assistance

Summarization

Question Answering


4. Evolution of Language Models

N-Gram → RNN → LSTM → Transformers

RNN

Processes sequential data

Suffers from vanishing gradient problem


LSTM

Advantages

Better memory retention

Handles long sequences


Disadvantages

Higher computational cost


Transformers

Advantages

Faster training

Better scalability

Improved context understanding


Attention Mechanism

Allows the model to focus on important words while processing text.
