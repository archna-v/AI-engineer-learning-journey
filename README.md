# 📅 Day 1: Foundations of AI and Machine Learning

## Data Collection
- Web Scraping
- APIs
- Existing Databases

### Key Principle
**Garbage In → Garbage Out**

Better quality data leads to better AI models.

## Types of Data
### Labeled Data
- Data with annotations
- Example: "Dog" vs "Not Dog"
- Used in supervised learning

### Unlabeled Data
- No predefined labels
- Used in unsupervised learning
- Faster and cheaper to collect

## Metadata
Metadata is data that describes other data.

Examples:
- Author
- Date
- File Size
- Usage Information

## Machine Learning
Machine Learning enables systems to learn patterns from data and make predictions.

### Types of ML
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

# 📅 Day 3: NLP, LLMs, Transformers & AI Ecosystem

## Natural Language Processing (NLP)
Allows computers to understand and generate human language.

Applications:
- Translation
- Summarization
- Chatbots
- Question Answering

## Language Models

### Masked Language Models
Predict missing words using surrounding context.

### Autoregressive Models
Predict the next word based on previous words.

## Large Language Models (LLMs)
Capabilities:
- Content Generation
- Coding Assistance
- Summarization
- Question Answering

## Evolution of Language Models
N-Gram → RNN → LSTM → Transformers

### Transformer Advantages
- Attention Mechanism
- Better Scalability
- Faster Training
- Improved Context Understanding

## LLM Development Lifecycle
1. Model Design
2. Dataset Engineering
3. Pre-training
4. Evaluation
5. Post-training
6. Fine-tuning
7. Final Testing

## AI Challenges
- Hallucinations
- Inconsistency
- Latency
- Data Limitations

## APIs
Applications communicate with AI models through APIs.

Client → API → AI Model → Response

## Vector Databases

- Video
